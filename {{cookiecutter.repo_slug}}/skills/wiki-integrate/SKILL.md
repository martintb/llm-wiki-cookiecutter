# Wiki Integrate Skill

Use this skill when the user wants `wiki/sources/` material connected to the rest of the wiki.

## Workflow

1. Read the relevant source summaries.
2. Update existing concept, project, synthesis, decision, person, organization, dataset, or question pages when possible.
3. Create new pages only when the topic is durable and recurring.
4. Preserve evidence trails back to source summaries.
5. Log what changed in `wiki/logs/maintenance.md`.

## Quality bar

- Prefer updates over duplicates.
- Distinguish source-supported fact from interpretation.
- Keep source trails explicit.
- Record contradictions instead of smoothing them over.
